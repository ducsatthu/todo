# ToDo-List

ToDo list is a simple web application 

### Features
* Pick a date from callendar
* Add new work
* Update work
* Edit work
* Delete work

### Technologies used:
* HTML5
* SCSS
* VueJs, Vuetify
* PHP 7.2

### Require Server

* Only support with domain or sub domain (not support alias or directory)
* PHP 7.2 or later
* For developer: NodeJs >= 12.8
